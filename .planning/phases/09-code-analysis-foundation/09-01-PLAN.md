Create `internal/code/goparser/` package with three files:

**patterns.go** — Detection pattern table:
- `DetectionPattern` struct: ImportPath, Function, Kind (detection_kind), TargetType, Confidence, ArgIndex (which arg has the target URL/connection string, -1 if none)
- `DefaultPatterns []DetectionPattern` covering:
  - HTTP: net/http Get/Post/NewRequest (0.9, service, http_call, argIndex 0 for Get/Post, 1 for NewRequest)
  - DB: database/sql Open (0.85, database, db_connection), github.com/jmoiron/sqlx Connect/Open, gorm.io/gorm Open, go.mongodb.org/mongo-driver/mongo Connect
  - Cache: github.com/redis/go-redis/v9 NewClient, github.com/go-redis/redis/v8 NewClient, github.com/bradfitz/gomemcache/memcache New (0.85, cache, cache_client)
  - Queue: github.com/IBM/sarama NewSyncProducer/NewAsyncProducer (queue_producer), NewConsumer/NewConsumerGroup (queue_consumer), github.com/rabbitmq/amqp091-go Dial, github.com/nats-io/nats.go Connect, github.com/aws/aws-sdk-go-v2/service/sqs NewFromConfig (0.85, message-broker or queue)
- `buildPatternIndex(patterns []DetectionPattern) map[string]DetectionPattern` — builds lookup map keyed by "importPath.Function"
- Comment hint patterns: regex matching `// (Calls|Depends on|Uses|Connects to) (\S+)` with confidence 0.4

**parser.go** — GoParser implementing LanguageParser:
- `GoParser` struct with patterns index (map[string]DetectionPattern)
- `NewGoParser() *GoParser` — builds pattern index from DefaultPatterns
- `Name() string` returns "go"
- `Extensions() []string` returns [".go"]
- `ParseFile(filePath string, content []byte) ([]CodeSignal, error)`:
  1. Skip if filePath ends with `_test.go` — return nil, nil
  2. Parse with `parser.ParseFile(fset, filePath, content, parser.ParseComments)`
  3. Build import alias map: local name -> full import path (handle renamed imports via imp.Name)
  4. Walk AST with `ast.Inspect` matching `*ast.CallExpr` -> `*ast.SelectorExpr` -> `*ast.Ident`
  5. Resolve ident.Name to import path via alias map
  6. Look up "importPath.Function" in pattern index
  7. Extract target from args: if pattern.ArgIndex >= 0, try to get `*ast.BasicLit` string arg, parse URL to extract hostname. Fall back to generic name from import path (last element) or pattern TargetType
  8. Scan comments for dependency hints using regex
  9. Return accumulated []code.CodeSignal
- `extractURLHost(call *ast.CallExpr, argIndex int) string` — extracts hostname from URL string literal args
- `evidenceSnippet(content []byte, lineNum int) string` — returns the source line at lineNum (trimmed, max 200 chars)

**parser_test.go** — Comprehensive tests using inline Go source as byte slices:
- Test HTTP detection: `http.Get("http://payment-api:8080/pay")` -> signal with target "payment-api", kind "http_call", confidence 0.9
- Test DB detection: `sql.Open("postgres", "postgres://primary-db:5432/mydb")` -> signal with target "primary-db", kind "db_connection", confidence 0.85
- Test cache detection: `redis.NewClient(&redis.Options{})` -> signal with target "redis" (generic), kind "cache_client", confidence 0.85
- Test queue detection: `sarama.NewSyncProducer(brokers, config)` -> signal with kind "queue_producer", confidence 0.85
- Test renamed import: `import pg "database/sql"` then `pg.Open(...)` -> detected correctly
- Test _test.go skip: file named "foo_test.go" returns nil signals
- Test import-only (no function call): file with `import "database/sql"` but no sql.Open -> zero signals
- Test comment hint: `// Calls payment-api` -> signal with target "payment-api", kind "comment_hint", confidence 0.4
- Test URL target extraction: various URL formats extracting hostnames
- Test variable arg fallback: `sql.Open("postgres", connStr)` where connStr is a variable -> falls back to "postgres" as target
