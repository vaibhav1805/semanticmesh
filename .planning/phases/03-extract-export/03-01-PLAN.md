Create `graphmdignore.go` with:
- `DefaultIgnorePatterns() []string` — returns sensible defaults: vendor, node_modules, .git, __pycache__, .venv, dist, build, target, .gradle, .next, out, .cache, bin, obj, .bmd, .planning
- `LoadGraphmdIgnore(dir string) (dirs []string, files []string, err error)` — reads `{dir}/.graphmdignore` file if it exists. One pattern per line. Lines starting with `#` are comments. Blank lines skipped. Patterns ending with `/` are directory patterns (added to IgnoreDirs), everything else is file patterns (added to IgnoreFiles). If file doesn't exist, returns DefaultIgnorePatterns for dirs and empty for files (no error).
- `GenerateGraphmdIgnore(dir string) error` — creates a `.graphmdignore` file with default patterns if one doesn't already exist. Include comments explaining the format.

Create `aliases.go` with:
- `AliasConfig struct` with `Aliases map[string][]string` field (canonical name -> list of alias names)
- `LoadAliasConfig(dir string) (*AliasConfig, error)` — reads `{dir}/graphmd-aliases.yaml` if it exists. Uses `gopkg.in/yaml.v3` for parsing (already in go.mod). Returns empty config (not error) if file doesn't exist.
- `(ac *AliasConfig) ResolveAlias(name string) string` — given a name, returns the canonical name if it matches any alias, otherwise returns the input unchanged. Build a reverse lookup map on first call (lazy init, stored in unexported field).
- `(ac *AliasConfig) ResolveAliases(names []string) []string` — batch resolve.

Tests for both:
- `graphmdignore_test.go`: Test loading existing file, missing file (defaults), comments/blank lines, dir vs file pattern classification, GenerateGraphmdIgnore creates file
- `aliases_test.go`: Test loading YAML, missing file, ResolveAlias for known alias, unknown name passthrough, case-sensitive matching
